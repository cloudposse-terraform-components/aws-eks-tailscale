## Usage

**Stack Level**: Regional

Use this in the catalog or use these variables to overwrite the catalog values.

```yaml
components:
  terraform:
    eks/tailscale:
      vars:
        enabled: true
        name: tailscale
        create_namespace: true
        kubernetes_namespace: "tailscale"
        image_repo: tailscale/k8s-operator
        image_tag: unstable
```
